# ml_vu_2026
This is our repo of the 2026 VU Machine Learning course project. It includes all the code and data necessary to replicate our comparitive study of synthetic data (minority class) oversampling- and hypertuning methods. The structure is as follows:

-  "XYZ_search.ipynb" files contain the code for each hypertuning method. The files are very similar, but were split because my pc with very _dated_ specs couldn't handle the length of the combined notebook.
-  "data.csv" for the dataset used.
-  "misc" contains old stuff that was saved because it could be useful. Provides some timeline as to how the project developed as I was not consistent with my commits.
